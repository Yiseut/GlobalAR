# Generated Stereo Instrumental Bed

This project uses a locally generated instrumental audio bed.

- Source: generated by `scripts/generate-audio.mjs` with local JavaScript synthesis.
- Third-party samples: none.
- Intended use: royalty-free background music for this local dashboard demo.
- Duration: 60 seconds.
- Style: soft corporate ambient with pad chords, low pulse, and light arpeggio.

Because the audio is synthesized locally from primitive waveforms and does not
sample an existing recording, it is safe to reuse as the default demo bed.
